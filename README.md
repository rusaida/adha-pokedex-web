# API'S

### Pokeapi

```js
https://pokeapi.co/api/v2/pokemon/${pokemon}
```

### Images of pokemons

```js
https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/other/official-artwork/${pokemon}.png
```
